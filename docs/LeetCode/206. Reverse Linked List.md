## 分析
```
  1 -> 2 -> 3
  最终到 2 的时候
  last = 3
  head = 2
  
```

## 题解

```go
/**
 * Definition for singly-linked list.
 * type ListNode struct {
 *     Val int
 *     Next *ListNode
 * }
 */
func reverseList(head *ListNode) *ListNode {
  if head == nil || head.Next == nil {
    return head
  }

  last := reverseList(head.Next)
  // last.Next = head 注意理解这里为啥不行
  head.Next.Next = head
  head.Next = nil
  return last
}
```
